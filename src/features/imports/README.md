# Imports Feature

Importación masiva de productos desde CSV.

## Estructura

- **components/** - Componentes UI (upload, resumen)
- **api/** - Lógica de API routes
- **lib/** - Parser CSV y helpers
- **schemas/** - Validaciones Zod para CSV

## Funcionalidades

- Upload de archivos CSV
- Validación de datos
- Creación/actualización masiva
- Reporte de errores
