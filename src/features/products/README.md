# Products Feature

Gestión completa de productos del showroom.

## Estructura

- **components/** - Componentes UI (tabla, dialogs, filtros)
- **api/** - Lógica de API routes
- **lib/** - Funciones helper y utilidades
- **schemas/** - Validaciones Zod

## Funcionalidades

- CRUD de productos
- Búsqueda y filtrado
- Paginación
- Ordenamiento por columnas
