# QR Feature

Generación y gestión de códigos QR para productos.

## Estructura

- **components/** - Componentes UI (dialog de generación)
- **api/** - Lógica de API routes
- **lib/** - Helpers para generar QR y subir a Cloudinary

## Funcionalidades

- Generación de QR codes
- Upload a Cloudinary
- Descarga de QR
- Impresión de QR
